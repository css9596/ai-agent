기존 게시판 시스템에 파일 첨부 기능을 추가해야 합니다.

요구사항:
1. 파일 첨부 기능 추가
   - 게시글 작성/수정 시 파일 첨부 가능
   - 파일 드래그 앤 드롭 지원

2. 파일 형식 및 용량 제한
   - 허용 확장자: jpg, png, pdf, xlsx
   - 파일당 최대 10MB
   - 게시글당 최대 5개 파일

3. 파일 관리
   - 목록/상세 화면에서 첨부 파일 표시
   - 첨부파일 개별 삭제 기능
   - 다운로드 기능

4. 관리자 기능
   - 첨부파일 용량 제한을 설정 페이지에서 변경 가능
   - 통계 및 모니터링 기능

5. 보안 및 성능
   - 파일 검증 (확장자, 크기, 내용)
   - 경로 조작 공격 방지
   - N+1 쿼리 최적화
